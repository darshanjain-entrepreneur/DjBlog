## Run Locally

Clone the project

```bash
  git clone https://github.com/darshanjain-entrepreneur/DjBlog.git
```

Go to the project directory

```bash
  cd djBlog
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, Material UI , Axios , Redux   

**Server:** Node, Express ,
mongoose , nodemon , cors ,dotenv , bcrypt .
